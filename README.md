# Comandante de Defensa Intergalactica

Aplicacion web offline para intervencion psicopedagogica. El foco es lectura comprensiva y ejecucion de instrucciones simples con numeros y cantidades.

## Como usar

1. Abra el archivo `index.html` en un navegador moderno (Chrome, Edge, Firefox).
2. Pulse **Comenzar**.
3. Lea la instruccion en voz alta, presione **ENTENDI**.
4. Arrastre los iconos a las casillas (o toque el icono para ponerlo).
5. Cuando este listo, pulse **¡DISPARAR!**.
5. Use **Pausa** para detener la sesion en cualquier momento.

## Ajustes rapidos

- Cambiar instrucciones y dificultad: edite `elements.js`.
- Cambiar estilos y colores: edite `style.css`.
- Ajustar logica, puntajes y validaciones: edite `game.js`.

## Sugerencias pedagogicas

- Reforzar el esfuerzo: el juego ya usa mensajes positivos.
- Pausar para dialogar cuando aparezca una instruccion nueva.
- Al final, revisar logros con el paciente (pantalla de cierre).

## Solucion de problemas

- Si no hay sonido, revise el boton **Sonido** en la parte superior.
- Si el navegador bloquea el audio, haga un clic en la pantalla y vuelva a intentar.
